The stochastic block model generator.

A degree-corrected micro-SBM via graph_tool.generate_sbm, walked frame by frame on the shared 20-node example.

SBM is the thinnest generator in the repo. Profile the input, hand its block matrix and degree sequence to graph-tool's micro degree-corrected sampler, then strip out the self-loops and parallel edges the micro constraint leaves behind.

Outliers fold into one __outliers__ block under the default combined mode. The three SBM-family visualizations (sbm, ec-sbm-v1, ec-sbm-v2) all share this 20-node input so the pipelines can be compared at a glance.
