ABCD, step by step

Config-model hybrid on empirical degrees and cluster sizes, with a global mixing parameter.

This walks through one default-settings run of ABCD (n_outliers = 0, is_CL = false, is_local = false) on a 20-node toy graph. The input is only used to extract three numbers: per-node degrees, per-cluster sizes, and the global mixing fraction ξ. Everything downstream is resampled from scratch.

Output clusters happen to line up with input clusters because we hand the sampler the same cluster sizes. The edges inside and across those clusters are new.
