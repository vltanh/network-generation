ABCD+o

ABCD with an explicit outlier mega-cluster: sampling step by step at default settings (outlier_mode=singleton, drop_oo=True, n_outliers=2).

This walkthrough uses the shared 20-node synthetic graph with three real clusters (C1, C2, C3) and two outlier nodes (19, 20). ABCD+o only sees aggregated statistics: a degree sequence, the real cluster sizes, a mixing parameter, and the outlier count. Sampling happens inside the Julia ABCD sampler, with a few careful modifications that the Python wrapper orchestrates around it.
