EC-SBM v1

A four-stage walkthrough at default settings: outlier mode forced to excluded, match-degree algorithm greedy.

EC-SBM v1 lifts plain SBM so every output cluster stays at least as edge-connected as the input. It runs four stages with seed offsets seed / seed+1 / seed+2: a profile that forces the excluded outlier transform, a clustered SBM with a K_{k+1} core, a separate outlier SBM, and a heap-greedy match-degree pass.
